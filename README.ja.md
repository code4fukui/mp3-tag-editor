# mp3-タグエディター

CLIで動作するMP3タグエディターです。

## デモ
```sh
deno -A example.js
```
- 試聴曲: [河和田小学校のテーマ](https://fukuno.jig.jp/4727) CC BY taisukef

## 機能
- MP3ファイルのタグ編集

## 必要環境
- Deno

## 使い方
1. 必要なライブラリをインストール
2. `example.js`を実行

## データ
サンプルデータには以下が含まれています:
- `theme-of-kawada-elementary-school.lyrics.csv`: 歌詞のCSVファイル
- `theme-of-kawada-elementary-school.lyrics.txt`: 歌詞のテキストファイル
- `theme-of-kawada-elementary-school.jpg`: アルバムジャケット画像
